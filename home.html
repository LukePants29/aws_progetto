AWS

AWS (Amazon Web Services) è una piattaforma di cloud computing
fornisce un'ampia gamma di servizi come storage, potenza di calcolo e gestione di database.

Docker

Docker è una piattaforma software che consente agli sviluppatori
di impacchettare le proprie applicazioni e dipendenze in container,
che possono essere facilmente distribuiti ed eseguiti su qualsiasi infrastruttura.

PHP

PHP (acronimo di Hypertext Preprocessor) è un linguaggio di scripting
adatto alla creazione di siti web dinamici. È uno dei linguaggi di
programmazione più utilizzati per lo sviluppo web, poiché può essere
integrato direttamente nel codice HTML.PHP viene eseguito sul server
e genera pagine dinamiche, che contengono contenuti personalizzati e
interattivi per ogni utente.

HTML

HTML (HyperText Markup Language) è un linguaggio di markup utilizzato
per pagine web. Consiste in una serie di tag o elementi che descrivono
la struttura e il contenuto di una pagina web. Gli elementi HTML
vengono utilizzati per definire il testo,le immagini, i link, gli
elenchi, le tabelle e altre forme di contenuto presenti nelle pagine
web. HTML viene interpretato dai browser web per visualizzare la pagina
web sullo schermo.

Bootstrap

Bootstrap è un popolare framework CSS che consente agli sviluppatori di
progettare e creare rapidamente siti Web reattivi. Fornisce un'ampia gamma
di componenti predefiniti come pulsanti, moduli e barre di navigazione che
possono essere facilmente personalizzati per soddisfare le esigenze del progetto.
