AWS

AWS (Amazon Web Services) è una piattaforma di cloud computing
fornisce un'ampia gamma di servizi come storage, potenza di calcolo e gestione di database.

Docker

Docker è una piattaforma software che consente agli sviluppatori
di impacchettare le proprie applicazioni e dipendenze in container,
che possono essere facilmente distribuiti ed eseguiti su qualsiasi infrastruttura.

PHP

PHP è un linguaggio di scripting lato server comunemente utilizzato
per creare applicazioni Web dinamiche

HTML

HTML è un linguaggio di markup utilizzato per strutturare e visualizzare i contenuti sul Web.

Boostrap

Bootstrap è un popolare framework CSS che consente agli sviluppatori di
progettare e creare rapidamente siti Web reattivi. Fornisce un'ampia gamma di
componenti predefiniti come pulsanti, moduli e barre di navigazione che possono
essere facilmente personalizzati per soddisfare le esigenze del progetto.
